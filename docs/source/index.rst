Welcome to Rebyu's documentation!
===================================

**Rebyu** is a Review Analysis Library equipped with text processing,
transformers, and diverse analyses like sentiment, topics, emotion,
and named-entities. It plans to be language-flexible, highly scalable,
and offers both preset pipelines and customizable options.

Version 0.1.6 (Released)

.. note::

   This project is under active development. Open contributions are accepted through PRs.
